# 🕵️‍♂️ Stalk Brawl Stars - Scraper + Case

Um scraper leve com integração de case pronta para bots WhatsApp que permite **stalkear jogadores de Brawl Stars** usando apenas a tag do jogador.

---

### 🔍 Funcionalidades

- Coleta dados do jogador diretamente do site **brawltime.ninja**
- Obtém:
  - Nome
  - Tag
  - Clube
  - Troféus atuais
  - Troféus máximos
  - Nível de experiência (EXP)
  - Foto de perfil

---

### ⚙️ Tecnologias utilizadas

- `Node.js`
- `axios`
- `cheerio`
